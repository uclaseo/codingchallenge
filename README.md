# Project Title

Coding Challenge

## Getting Started

All scripts are in Javascript(Node).

### Prerequisites

Install dependencies.
```
npm install
or
yarn install
```
Create API_KEY.js in the root folder, and add the following:
```
const API_KEY = 'API KEY HERE';
module.exports = API_KEY;
```

## Running the tests

Run test for findArrayDimension.js
```
npm test
or
yarn test
```
